- Given application uses Vuforia and Unity.
- Given 2 target images, it generates 1 witch and 1 dragon charactors.
- if dragon and witch charactors are close (closer than 100 units) they will fight.
- if they are further than this threshold, they will walk towards each other.

To install this app on your android device, copy measuARwar.apk and install it in the device.
